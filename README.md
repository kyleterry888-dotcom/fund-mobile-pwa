# 自用基金估值助手 Mobile PWA

这是手机自用版。它是纯静态网页，不需要后端、不需要数据库、不需要 API Key。

## 怎么运行

本地测试：

```bash
cd fund_mobile_pwa
python -m http.server 8010
```

浏览器打开：

```text
http://localhost:8010
```

手机使用时，建议部署到 HTTPS 静态网页服务，例如 Cloudflare Pages、GitHub Pages、Vercel、Netlify 或自己的服务器。

## iPhone 添加到主屏幕

1. 用 Safari 打开你部署后的网址
2. 点底部“分享”按钮
3. 选择“添加到主屏幕”
4. 桌面会出现“基金估值”图标

## 安卓安装

用 Chrome 打开后，通常会出现“安装”或“添加到主屏幕”。也可以点页面里的“安装”。

## 数据保存在哪里

基金列表、份额、成本、备注、手动重仓都保存在当前浏览器的 localStorage。

换手机、换浏览器、清理 Safari 网站数据之前，请先用“导出备份”。

## 估值逻辑

- 平台估值：读取天天基金估值接口
- 最新净值：读取东方财富历史净值
- 自动重仓：读取东方财富 F10 前十大持仓
- 个股行情：读取腾讯行情
- Top10 贡献：Σ 重仓权重 × 个股涨跌
- Top10 归一化：Top10 贡献 / Top10 覆盖率

## 注意

公开接口可能会因为跨域、限流、接口变更而偶尔失败。这个工具适合自用估算，不适合作为真实交易依据。
